# Getting Started
To get started locally follow these simple example steps.
- Fork and clone the repo to your computer using git clone `https://github.com/sportypro/jwplayer_video.git` in the terminal or command prompt.
- Create a new feature branch using `git checkout -b feature-name`.
- Install the dependencies `npm run install`.
- Run project locally with `ionic cap run android -l --external --source-map=false`.
